Web Design Game

Created by Sha, Neutronito & Soul

Description & Purpose

  • The Web Design Game's purpose is to provide individuals with an interactive, fun & professional environment to acquire and practice web design skills including html/css/js
  • This is achieved by allowing users to complete daily challenges, which are in the form of webpage images/maps, and the aim of the user is to replicate the given image/map using web design skills.
  • The daily maps are uploaded by the admin user resulting in an arbitrary number of maps of various difficulties allowing the game to be used by educational institutions as well as individuals
  • This in turn allows the game to be used for all knowledge levels of web design, and due to the high level of user authentication and secure data storage and retrieval practices makes the game secure and reliable
  • Additionally being able to compete with other users, or simply comparing results is allowed due to an extensive leaderboard system with various sorting mechanisms and time intervals

Instructions

  1. Before playing the game the user should Login/Signup with their account following the navbar links
  2. After successful authentication, the user should select the "Game" tab and press the "Show Map" button. This will display the current daily challenge
  3. The code should be written inside the text box area, and if at any moment the user wishes to preview what their website/map would look like, pressing the "Preview Map" button will display it
  4. Once the user is satisfied with the similarity of their solution with the given map/image, pressing the "Submit" button will submit the users code and provide them with a score depicting the similarity of their solution with the provided map/image
  5. The users score is automatically entered in the database so selecting the "LeaderBoard" button allows the user to view their score in comparison to other users
  6. If the user wishes to share their score, pressing the "Share Score" button will copy their achieved score to the clipboard
  7. The "head start" button will simply insert a generic HTML template, including a CSS section. This is just an ease of life feature if you do not wish to fill in all those minor details yourself.
