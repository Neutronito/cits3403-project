Welcome to the Button Pushing Homepage!

This is a website we have created to develop our website development abilities. Its purpose is to allow the user to press the button on screen as many times as they want, and have the number of presses recorded. Additionally, they will have the ability to create an account with an associated username and password. The user will then be able to log into this account and record how many times they have pressed the button since the account creation. The idea behind is that we will have to create a back-end process that will recieve the number of times you press the button in one sitting and store this number, which you can then view after you login. This site will also have a leaderboard to see who has pressed the most times.
